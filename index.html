    Boost Gaming

    Optimización completa en un toque. Libera RAM, activa rendimiento, prioriza GPU y refresco máximo.

    Liberar RAM

    Vacía caches del sistema y mata procesos no esenciales (no afecta tus juegos).

    shizuku exec "am send-trim-memory PRUNE_LEVEL_RUNNING_MODERATE"

    Forzar Performance

    Sobrescribe power.save y activa el governor de máximo rendimiento.

    su -c "echo performance > /sys/devices/system/cpu/cpu0/cpufreq/scaling_governor"

    Refresh Rate Max

    Establece la tasa de refresco al máximo soportado por el panel.

    shizuku exec "settings put system user_rotation 0 && settings put global peak_refresh_rate 144"

    Desactivar Animaciones

    Reduce latencia visual poniendo las animaciones a 0.5x.

    shizuku exec "settings put global window_animation_scale 0.5; settings put global transition_animation_scale 0.5; settings put global animator_duration_scale 0.5"

    Congelar Apps

    Pausa apps en 2º plano usando Brevent para reducir uso de CPU.

    brevent list -freeze com.example.app

    DNS Privado

    Reduce latencia en juegos online con DNS optimizado.

    shizuku exec "settings put global private_dns_mode hostname; settings put global private_dns_specifier dns.google"

    Touch Boost

    Aumenta la sensibilidad táctil y velocidad de respuesta.

    shizuku exec "settings put secure touch_event_threshold 4; settings put system pointer_speed 7"

    Thermal Performance

    Sobrescribe límites térmicos con thermal HAL cuando está disponible.

    shizuku exec "cmd thermalservice override-status 0"
